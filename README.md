## 📌 Sobre o Projeto
Este espaço armazena os exercícios, projetos e materiais desenvolvidos durante as aulas. O foco é o aprendizado de lógica de programação, POO (Programação Orientada a Objetos) e boas práticas com a linguagem Java.

## 🚀 Tecnologias e Ferramentas
*   **Linguagem:** Java (JDK 17+)
*   **IDE:** IntelliJ IDEA / Eclipse / VS Code (escolha a sua)
*   **Paradigma:** Orientação a Objetos

## 📂 Estrutura do Repositório
*   `/exercicios`: Listas de exercícios práticos.
*   `/projetos`: Trabalhos e projetos avaliativos.
*   `/anotacoes`: Resumos e notas de aula.
